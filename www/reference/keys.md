| Key | Action |
| --- | --- |

| <span class="pitwall-key">tab</span> | Cycle between tables on two-table screens |
| <span class="pitwall-key">v</span> | Live screen: cycle battle views (all → lead → podium → points) |
| <span class="pitwall-key">1</span>–<span class="pitwall-key">5</span> | Game wizard: tyre compounds (S/M/H/I/W) |

| <span class="pitwall-key">enter</span> | Game wizard: confirm step |
| <span class="pitwall-key">backspace</span> | Game wizard: undo |
